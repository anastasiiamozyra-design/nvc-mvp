# NVC Practice Tool

This project is a simple web application designed to support the practice of Nonviolent Communication (NVC) based on Marshall Rosenberg’s model.

## Purpose
The goal of this tool is to provide a structured way to practice communication scenarios and reflect on responses in everyday situations.

## Use Case
The application was developed in response to a real request from a user interested in practicing NVC after completing a course. It serves as a simple interactive format for ongoing practice.

## Features
- Input-based interaction for communication practice  
- Response generation and reflection support  
- Simple and accessible interface  

## Tech Stack
- Next.js  
- API routes for text processing
